# 🚔 Crime Analysis Dashboard (India)

An interactive and data-driven **Power BI dashboard** designed to analyze crime patterns across India. This project transforms raw crime data into meaningful insights using visualization and analytical techniques.

---

## 📌 Project Overview

This project focuses on analyzing crime trends across different states and years in India. The dashboard provides insights into crime distribution, growth trends, and key performance indicators to support better understanding and decision-making.

---

## 🎯 Objectives

- Analyze crime trends over time  
- Compare different crime categories  
- Identify top crime-contributing states  
- Understand distribution of crimes across regions  
- Provide interactive filtering for better insights  

---

## 📊 Features

- 📍 State-wise crime analysis (Map Visualization)  
- 📈 Year-over-Year crime trend analysis  
- 📊 Crime category comparison (Murder, Theft, Rape, Robbery)  
- 🏆 Top 5 crime states identification  
- 📉 Growth rate analysis  
- 🎛️ Interactive filters (Year, State, Metrics)  
- 📄 Multi-page dashboard (Overview & Insights)  

---

## 🛠️ Tools & Technologies

- **Power BI** – Dashboard & Visualization  
- **Microsoft Excel** – Data Cleaning & Preprocessing  
- **DAX** – Measures & Calculations  

---

## 📂 Dataset

The dataset is sourced from:

- **NCRB (National Crime Records Bureau), Government of India**

---

## 🔄 ETL Process

### 1. Extract
- Data collected from NCRB datasets  
- Includes state-wise, year-wise, and category-wise crime data  

### 2. Transform
- Data cleaning (null values, duplicates removal)  
- Data formatting and filtering  
- Creation of new columns (YoY Growth, Total Crimes)  

### 3. Load
- Data loaded into Power BI  
- KPIs and relationships created  
- Interactive dashboards developed  

---

## 📊 Key Insights

- Theft is the most dominant crime category  
- Crime trends show fluctuations over the years  
- Certain states contribute significantly higher crime rates  
- Growth patterns indicate socio-economic influence on crime
